Deep Learning for Houdini

  • This application is used to predict parameters from images of furniture arrangements, and create 3D models of floorplans in Houdini
  • Training images and data was created in Houdini using the PDG Workflow (Wedge render node).
  • AI models were created and trained using MobileNet.
  1. Flask is used to accept local GET requests with the input image name.
  2. When the Python Houdini node is made visible, it triggers the Flask application running in the background.
  3. This, in turn, triggers a Python script, that uses trained models to predict parameters from the input image.
  4. The predicted values are plugged back into the Houdini geometry network
